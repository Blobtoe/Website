For Loops

A for loop is a section of code that runs over and over a certain amount of times. This is the syntax for a for loop:


for (init; condition; increment) {
    code to be run multiple times...
}

The init step is executed first, and only once. This step allows you to initialize a variable for loop control.

Then the condition is evaluated. If the condition is evaluates to true, the code in the loop runs. If it evaluates to false, the program skips the code in the loop.

After the code in the loop is run, the increment statement is executed. This statement allows you to update the loop control variable.

Next, the condition is evaluated again, the loop starts again.

Example:

for (int i = 0; i < 5; i = i + 1) {
    code to be run multiple times...
}

In this example, the code in the loop would be run 5 times.
