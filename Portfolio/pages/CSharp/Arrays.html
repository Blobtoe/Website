Arrays

An array is a list, or collection of elements of the same type. Instead of creating individual variables, an array can be used.


Initializing Arrays

Arrays can be initialized with the following syntax:
datatype[] arrayName = new datatype[x];
where datatype is the type of data the array will be containing, arrayName is the name of your array, and x is the number of elements your array will be containing (the size of the array).


Accessing Array Elements

Array elements can be accessed with the following syntax:
arrayName[x]
where arrayName is the name of your array and x is the index of the element you want to access. Remember array's indexes are 0-based, meaning the index of the the first element is 0 and so on.
