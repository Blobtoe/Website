If Statements

An if statement is a a section of code that only runs if a certain requirements are met.


Syntax

The syntax for an if statement in C# is:

if (boolean_expression) {
    code...
}

where boolean_expression is an expression that evaluates to true or false. The code in the curly braces will only run if the expression evaluates to true.


Else If and Else Statements

Else if and else statements come after an if statement or another if else statement. They are used to evaluate various conditions and have different outcomes.

if (boolean_expression_1) {
    code...
}
else if (boolean_expression_2){
    code...
}
else {
    code...
}
