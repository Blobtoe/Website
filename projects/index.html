RECEIVING IMAGES FROM WEATHER SATELLITES

Using an RTL-SDR, a USB device that receives radio waves, I was able to receive and decode signals sent from the NOAA and METEOR weather satellites and produce images. I had to build a special antenna specifically tuned to the satellites' frequencies. After doing everything manually, I moved to automating the process. I wrote my own software to record and decode the satellites as they passed overhead, then post them on my website. I also made an app to decode images on my phone.

SMART LIGHT

Using a 5v relay, I am able to control my desk lamp form my raspberry pi. This allowed me to control my light depending on some sensors and triggers. I connected a movement sensor to my raspberry pi, so it can automatically toggle my light depending on if there is movement in my room. Using python and flask, I setup a server which allows me to control my light by sending it a command via the internet. This allows me to toggle my light from my phone, or from google assistant.
